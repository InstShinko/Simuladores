# Simulador Académico

Plataforma de simulación para prácticas educativas con múltiples materias y trimestres.

## Materias disponibles
- Ciencias Sociales I (completo: 9 actividades, 90 preguntas)

## Tecnologías
- HTML5, CSS3, JavaScript
- Firebase Authentication
- Firebase Hosting (opcional)

## Cómo usar
1. Inicia sesión con tu cuenta (creada por el administrador)
2. Selecciona trimestre, materia y actividad
3. ¡Comienza a practicar!
